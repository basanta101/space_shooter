# Space Shooter
A simple shooting game made with vue3 and vite


## Run Locally

Clone the project

```bash
  git clone https://github.com/basanta101/space_shooter.git
```

Go to the project directory

```bash
  cd to folder directory
```

Install dependencies

```bash
  npm install
```

Start the dev server

```bash
  npm run dev
```

## Tech Stack

**Client:** Vue.js, Vite
